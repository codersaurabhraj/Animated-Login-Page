# 🔐 Animated Login Page

A modern and responsive **Animated Login Page UI** built using HTML and CSS.  
This project features smooth animations, glassmorphism design, and a clean user interface.

---

## 🚀 Features

- ✨ Smooth animations  
- 🧊 Glassmorphism UI (blur effect using backdrop-filter)  
- 📱 Fully responsive design  
- 🔐 Clean and minimal login form  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3
